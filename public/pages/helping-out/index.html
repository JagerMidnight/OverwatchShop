Want to help out this Fan-Made Vercel Site?

This site showcases the Overwatch Shop and Hero Gallery, and we'd love your help making it even better!

No Coding Experience? No Problem!

If you've got an idea or found something that needs fixing but aren't sure how to code, just submit an issue on GitHub. A contributor will pick it up and handle the coding side of things. It's a fantastic way to contribute!

Got Coding Skills?

If you have some coding knowledge and want to dive in, feel free to create a pull request! Once your request is reviewed and looks good, we'll get it merged in. Your contributions are super valuable!

---

Understanding the Project Files (for PRs)

When you're submitting a pull request, understanding where things go can be a huge help. Here's a quick rundown of some key files and directories:

  • `_heroinfo/heroname.json`: Stores hero information as seen in-game as "Hero Information".
  • `assets/cosmetics/type/heroname.json`: Stores a list of "cosmetics" of a specific `type` for the specified hero. Example: `assets/cosmetics/skins/reaper.json`.
  • `vercel.json`: Vercel site configuration.
  • `assets/_heroicons/heroname.webp`: Contains a list of icons in a compressed and recommended format for web assets. Please make sure these are `.webp` files!
